# HR Analytics Dashboard

An interactive *HR Analytics Dashboard* built using *Power BI* to provide actionable insights into workforce trends, employee performance, and attrition patterns. This project demonstrates end-to-end data handling, from cleaning raw HR data to visualizing it in interactive dashboards.

---

## *Project Overview*

The goal of this project is to help HR teams make *data-driven decisions* by analyzing employee data. Key features include:

- Employee demographics analysis (department, location, age, gender)  
- Attrition trends and turnover analysis  
- Performance metrics and distribution  
- Interactive filters for time periods, departments, and locations  

---

## *Workflow*

1. *Data Collection & Cleaning*
   - Raw HR data collected in *CSV files* (employee details, attendance, performance, etc.)  
   - Data cleaned, transformed, and validated using *MySQL* to ensure accuracy and consistency

2. *Data Storage*
   - Cleaned data uploaded to a *MySQL database* for efficient querying and management

3. *Data Visualization*
   - Connected MySQL database to *Power BI*  
   - Built interactive dashboards to visualize HR metrics and trends  
   - Created dynamic charts, graphs, and filters to analyze workforce data

---

## *Technologies Used*

- *MySQL* – Data cleaning, transformation, and storage  
- *Power BI* – Interactive data visualization and dashboard creation  
- *CSV Files* – Initial raw HR data source  

---

## *How to Use*

1. Clone or download the repository  
2. Import your cleaned HR dataset into MySQL (optional if you want to replicate analysis)  
3. Open the Power BI dashboard file (.pbix) and connect it to your MySQL database  
4. Interact with charts, filters, and visuals to explore HR insights  

---

## *Key Benefits*

- Helps HR teams monitor workforce metrics efficiently  
- Supports data-driven decision-making  
- Provides clear visual insights into employee trends and performance
